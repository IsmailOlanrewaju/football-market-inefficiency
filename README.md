# football-market-inefficiency
## Project Scope

This study examines the relationship between player performance indicators and market valuation in the modern football transfer market.

Rather than attempting to create a universal player pricing system, the project uses the U-23 player market as a case study to investigate how closely transfer valuations align with measurable football performance.

The dataset consists exclusively of league performance data from the 2025/26 season across Europe's five major leagues:

- English Premier League (EPL)
- La Liga
- Bundesliga
- Serie A
- Ligue 1

The analysis focuses on U-23 players because this market segment is heavily influenced by future expectations, potential, and reputation, creating an interesting environment for studying valuation differences.
